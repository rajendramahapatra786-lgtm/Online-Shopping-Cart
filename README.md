# AnimeShop - Complete E-commerce Website

## Project Overview
A fully functional anime merchandise e-commerce website with cart, wishlist, user authentication, and category-based filtering.

## Features
- 🛍️ **Complete Shopping Cart** with localStorage
- ❤️ **Wishlist** functionality
- 🔍 **Search** across all products
- 👤 **User Authentication** (Login/Signup)
- 📱 **Fully Responsive** design
- 🎨 **Category Pages**: Fashion, Watches, Bags
- 👕 **Fashion Sub-categories**: T-Shirts & Hoodies with size selector
- 💰 **Price Formatting** in Indian Rupees (₹)
- ⭐ **Product Ratings** and reviews
- 🔄 **Sort & Filter** options

## Pages
1. **index.html** - Homepage with all products
2. **fashion.html** - Fashion category (Tshirts + Hoodies)
3. **watches.html** - Anime watches
4. **bags.html** - Anime bags
5. **cart.html** - Shopping cart
6. **wishlist.html** - Wishlist
7. **login.html** - Login/Signup
8. **checkout.html** - Order confirmation

## Technology Stack
- HTML5
- CSS3 (with Flexbox & Grid)
- Vanilla JavaScript
- localStorage for data persistence
- Font Awesome Icons
- CDN for images

## How to Run
1. Download all files maintaining the folder structure
2. Open any HTML file in a modern web browser
3. No server required - works directly!
